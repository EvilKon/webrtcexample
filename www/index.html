WebRTC + Erlang:
video web chat and 'chatroulette'

This is an example of using WebRTC:
"video conference in a browser".

Supported browsers:
Chrome (stable version)
Firefox (nightly version).

Try!

WebRTC

Video and audio Real-Time Communication for browsers. Available in Chrome's stable version, Firefox's nightly version.

JavaScript

Handles API to WebRTC in a browser, manages websockets connection to server, does some browser side work.

Bootstrap

Nice and simple web frontend.

Erlang

Works on server. Handles browsers' registration, video rooms, WebRTC signaling, web chat, websockets transport.

Cowboy

Nice high-performance web server written in Erlang. Handles websockets connections and some HTTP requests.

Websockets

Transport for signaling messages between browsers and server. Transport for clients' chat messages.
